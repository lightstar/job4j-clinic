# Клиника домашних питомцев

Мой вольный вариант программы **"Клиника домашних питомцев"** из курса Петра Арсентьева **"Java. Путь от ученика до эксперта"**.

Ссылка на сам курс: http://job4j.ru/courses/java_way_from_student_to_master.html

Для запуска (с учетом установленных JDK и maven) нужно ввести в директории проекта команду: `mvn exec:java`

Суть этой простой консольной программы в том, что имеется клиника домашних животных, и вводя различные команды можно
выполнять над ней простые действия, например, добавить клиента, установить ему животное, редактировать, удалять,
искать по имени животного и клиента. На данном этапе все данные хранятся в памяти и теряются после выхода из программы.
